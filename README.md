### 🛠️ O Arsenal (Missões Concluídas)
Dentro do sistema você vai esbarrar com os meus principais arquivos operacionais, como:
- **SCARLET SCANNER:** Ferramenta tática de reconhecimento (Port Scanner) feita em Python.
- **Parecer Técnico SS7:** Análise independente de vulnerabilidades e vetores de ataque em redes móveis (SS7/Diameter).
- **Ferramentas de Auditoria & Purple Team:** CLI tools de compliance e simuladores de emulação de adversários via MITRE ATT&CK.
- **Pesquisa PIBIC:** Estudo acadêmico focado no desenvolvimento de Inteligência Artificial ética e transparente.

### 💻 Como esse terminal foi forjado
Para sustentar toda a imersão de *Signalis*, a infraestrutura roda com:
- **React (Vite)**
- **Tailwind CSS** (Responsável pelo vermelho agressivo, os tons de ciano e a formatação de terminal)
- **Animações CSS** (Para as scanlines, efeitos de pulso e *hover* tático dos cards)
